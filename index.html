01. 基本の出力

3色のログを使い分けて、プログラムの動きを視覚化しましょう。

disp("こんにちは") // 通常(緑) cav("注意してください") // 警告(黄) ere("エラー発生") // エラー(赤) ent,名前 // ユーザー入力を変数に保存

02. 計算と判断 (if/else)

日本語変数を使って計算し、ifで判断を行います。最後は必ず}}で閉じます。

A = 10 B = 20 合計 = cal(A "+" B) if (合計 = 30) disp("正解！") }} else ere("ちがうよ") }}
※比較は「=」を1つだけ使います。

03. 応用：JS連携とコピー

JavaScriptのパワーを JS, で取り寄せ、icp で自由にデータを複製します。

++StartScript++ this.rand = Math.floor(Math.random() * 10); ++endscript++ disp(JS,rand) icp("""script""") // 全コードをコピー icp("""log""") // 全ログをコピー

04. ゲーム制作ガイド

計算クイズゲーム

ent で回答をもらい、if で判定する基本構成です。
disp("3 + 5 は？") ent,ans if (ans = 8) disp("正解！") }}

タイピングゲーム

keypush でキー入力を、{{ }} で時間制限を作ります。
disp("3秒以内に[Z]を押せ！") {{ "3" }} keypush"Z" { disp("成功！") }
